# Italy Justice System (2003–2024) — Data Analysis

Analisi esplorativa del sistema giudiziario italiano basata su dati ufficiali dal 2003 al 2024.

## Obiettivi
- Analizzare l’andamento dei procedimenti civili e penali
- Individuare trend e criticità del sistema
- Studiare l’evoluzione dei tempi e dei carichi di lavoro

## Dataset
Il progetto utilizza tre dataset principali:
- Procedimenti civili
- Procedimenti penali
- Indicatori di efficienza del sistema giudiziario

## Contenuto della repository
- Dataset in formato CSV
- Notebook Python con analisi, grafici e commenti

## Tecnologie utilizzate
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
